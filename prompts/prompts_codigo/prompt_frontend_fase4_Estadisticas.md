# Objetivo

En esta fase nos enfocamos en el "esqueleto" de la pantalla, el tema de colores y las tarjetas de datos. Necesito crear la base de una pantalla llamada StatsScreen.

1. Define un objeto de Theme con colores: Primary, Background, CardBackground y Text.  
2. Crea el **Header**: Un selector (Dropdown) para elegir el hábito (con la opción "Vista Global" por defecto).  
3. Crea los **Tabs**: Un componente de switch/tabs para seleccionar entre "Semanal", "Mensual" y "Total".  
4. Crea el componente **StatCard**: Debe recibir un título y un valor. Organízalos en una cuadrícula (Grid) de 2x2 para mostrar: Racha Actual, Racha Máxima, Tasa 7d y Tasa 30d.  
5. Estilo: Limpio y moderno, con buen uso de padding y sombras suaves para las tarjetas.

# Indicaciones adicionales

* Usa TypeScript estricto.  
* Crea metodos y ficheros con responsabilidades y tamaños razonables  
* Comenta bastante los metodos
