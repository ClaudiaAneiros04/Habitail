# Objetivo

Necesito crear la base de una aplicación de hábitos donde la protagonista es una mascota virtual (un gato). El objetivo de esta fase es dejar lista la estructura de datos, la gestión de assets y la pantalla principal.

**1\. Definición de Tipos/Interfaces:**

Crea una estructura de estado para la mascota que incluya:

* vida: numérico (0-100).  
* nivel: numérico.  
* xp: numérico.  
* xpParaSiguienteNivel: numérico.  
* estadoActual: un enum o unión de tipos con los valores: absent, sad, confused, cheering, happy.  
* skinEquipada: string (ID de la skin).  
* accesorios: array de IDs de accesorios equipados.
