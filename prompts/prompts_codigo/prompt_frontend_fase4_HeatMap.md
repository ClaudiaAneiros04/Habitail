# Objetivo

Aquí implementamos la visualización de 52 semanas usando la librería específica. Ahora, basándote en la StatsScreen anterior, vamos a añadir el **HeatmapCalendar** usando la librería react-native-chart-kit.

1. Usa el componente ContributionGraph.  
2. Configuración: 52 semanas x 7 días.  
3. **Colores:** Implementa una escala de 4 tonos de verde (o el color del tema) basados en la completitud: Sin datos, \<25%, \<75%, y 100%.  
4. **Eje X:** Debe mostrar los meses abreviados.  
5. **Interactividad:** Envuélvelo en un ScrollView horizontal para que el usuario pueda navegar por todo el año.  
6. Proporciona un set de datos de prueba (Mock Data) para visualizar cómo se llena el gráfico.
